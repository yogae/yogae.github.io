flask는 자동적으로 directory와 serving하는 곳에서  `flaskr/static`와 관련된 path를 가지는  `static` view를 추가합니다. 

CSS, JavaScript functions, login image와 같은 file이 `flaskr/static` directory 밑과 `url_for('static', filename='...')` 에 언급되어진 path에 존재해야합니다.
